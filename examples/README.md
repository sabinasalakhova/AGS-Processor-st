# Examples

This directory contains example scripts demonstrating how to use the AGS Processor.

## Available Examples

### basic_usage.py
Demonstrates basic AGS file processing with validation and export.

```bash
python examples/basic_usage.py
```

### batch_processing.py
Shows how to process multiple AGS files and consolidate them into a single output.

```bash
python examples/batch_processing.py
```

## Usage

1. Update the file paths in the examples to point to your AGS files
2. Run the example scripts:
   ```bash
   cd /path/to/AGS-Processor-st
   python examples/basic_usage.py
   ```

## Note

Make sure you have installed the AGS Processor before running the examples:

```bash
pip install -e .
```
